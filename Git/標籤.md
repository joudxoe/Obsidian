要點整理
- 標籤是一個指向某個Commit的指標

標籤類型
- 輕量標籤 (lightweight tag) - 個人使用或是暫時標記用途
- 附註標籤 (annotated tag) - 可用於做版本號之類的用途

---

```
$ git tag cats 51d54ff						// 建立一個輕量標籤
```

```
$ git tab cats 51d54ff -a -m 'cats'			// 建立一個附註標籤
```

```
$ git show cats								// 檢視標籤
```

```
$ cat .git/refs/tags/cats					// 檢視標籤文檔內容，內容為40個字元SHA-1值
```

```
$ git tag -d cats							// 刪除標籤
```
