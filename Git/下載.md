要點整理
- `pull`指令用於下載遠端數據庫的資料來更新/同步本地端的數據庫
- `git pull` = `git fetch` + `git merge`

---

```
# 將遠端拉回本機更新
$ git pull origin master

# ===

# 取得遠端提交的版本
$ git fetch origin

# 合併遠端的master分支
$ git merge origin/master
```

```
# 使用rebase方式進行合併
$ git pull origin master --rebase

# ===

$ git fetch origin
$ git rebase remotes/origin/master
```
