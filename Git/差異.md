要點整理
- `diff`指令可用於比對兩個版本之間的差異

---

```
# 查看Changes not staged for commit/工作目錄內全部檔案的修改內容
$ git diff
```

```
# 查看Changes not staged for commit內指定檔案的修改內容
$ git diff <file>
```

```
# 比對兩個版本之間的差異
$ git diff <old_commit> <new_commit>
```

```
# 檢視Changes to be committed/暫存區檔案修改的內容
$ git diff --cached
$ git diff --staged
```
