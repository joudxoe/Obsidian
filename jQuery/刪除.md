刪除操作
- `empty()` <small>刪除子元素及後代元素和所選元素的文字內容</small>
- `remove()` <small>刪除所選元素中的所有內容</small>
- `removeAttr()` <small>刪除所選元素的屬性</small>
- `unwrap()` <small>刪除所選元素的父元素</small>

---

```
$('.container').empty();
```

```
$('p.hint').remove();
```

```
$('p').remove('.hint');
```

```
$(this).unwrap();
```

```
$('a').removeAttr('href');
```
