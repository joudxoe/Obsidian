要點整理
- 在Console窗口輸入`$.fn.jquery`命令查看當前jQuery使用的版本
- 在Console窗口輸入`$().jquery`命令查看當前jQuery使用的版本

---

```
$().jquery;
```

```
$.fn.jquery;
```

```
<!--[if lt IE 9]>
	<script src="jquery-1.9.0.js"></script>		// jQuery 2不支持IE6,7,8舊版瀏覽器
<![endif]-->
```

```
<!--[if gte IE 9]>
	<script src="jquery-2.0.0.js"></script>
<![endif]-->
```
