循環語句/重覆/迴圈結構
- `while`
- `do...while`
- `for`
- `foreach` <small>用於遍歷數組和對象</small>

for
```
for ($i = 0; $i < 10; $i++) {

}
```

for...endfor
```
for ($i = 0; $i < 10; $i++) :

endfor;
```

foreach
```
foreach ($array as $key => $val) {

}
```

foreach...endforeach
```
foreach ($array as $key => $val) :

endforeach
```

while
```
while ($expr) {

}
```

while...endwhile
```
while ($expr) :

endwhile;
```

do...while
```
do {

} while ($expr);
```
