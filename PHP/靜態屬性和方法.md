*宣告類別屬性和方法為靜態(static)，可以不需產生實體類別直接存取*

*靜態屬性不需產生實體就可直接使用，類別名稱::靜態屬性名稱*

*靜態方法不需產生實體即可直接使用，類別名稱::靜態方法名稱*

*靜態屬性不能透過已產生實體類別的物件直接存取，但靜態方法可以*

*靜態方法不需透過物件即可呼叫，因此$this在靜態方法中不可用*

```
class guests{
	public static $name='Guest';
	public static getName(){
		return self::$name;
	}	
	echo guests::$name;
	echo guests::getName();
}
```
