條件語句/分支/選擇結構
- `if`
- `if...else`
- `if...elseif...else`
- `switch`

if...elseif...else
```
if ($expr1) {

} elseif ($expr2) {

} else {

}
```

if...endif
```
if ($expr1) :

elseif ($expr2) :

else :

endif;
```

switch
```
switch ($expr) {
	case 0:
		break;
	case 1:
		return;
	default:
		break;
}
```

switch...endswitch
```
switch ($expr1) :
	case 0:
		break;
	case 1:
		break;
	default:
		break;
endswitch;
```
