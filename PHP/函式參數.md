傳遞參數的方式
- 按值傳遞 (不會影響到實際參數的值)
- 引用傳遞 (==會影響到實際參數的值==)
- 默認參數/可選參數

參數類型
* ==形式==參數 - 定義函數時使用的參數
* ==實際==參數 - 調用函數時傳入的參數/變數/數據

自訂函式的參數預設值/默認參數
```
function showAuthor($name='Jane'){
	echo $name;
}
showAuthor();			// Jane
showAuthor('Jenny');	// Jenny
```

可變參數函數
```
function vars(...$args){
	$num=count($args);	// 統計參數個數
}
```

要點整理
- 設定形參就必須傳入實參，否則會報錯
- 調用函數若沒有傳遞參數，便會使用設定的默認參數
- 向函數傳遞變數參考，使用`&`符號
