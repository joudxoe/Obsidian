傳遞變數/值的方式
- 傳值 (Call By Value)
- 傳址 (Call By Reference)

變數引用
```
$name='Jane';
$author=&$name;
$name='Jenny';
echo $author;		// Jenny
$author='Sam';
echo $name;			// Sam
```

參數引用
```
function add(&$sum){
	//return $sum++;	
	return ++$sum;
}
$num=10;
echo add($num);		// 11
echo $num;			// 11
```

取消引用
```
unset($author);
```

要點整理
- 當不在需要引用時，可以取消引用，使用`unset()`函數
