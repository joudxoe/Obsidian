*定義/創建一個類別*
```
class Employee{
	var $name;
	protected $age;
	private $wage;
	function function checkIn(){
		echo $this->name.' check in at '.date('h:i:s');
	}	
}
```

*實例化類別*
```
$emp=new Employee();
```

*存取/訪問物件/對象的成員屬性和方法*
```
$emp->name='Jane';
$emp->checkIn();
```

***

類別中的常數

* 使用const關鍵字定義，不需定義存取範圍，一律為public
* 命名不需$符號
* 不需建立物件存取

*存取類別常數的方式*

* 類別名稱::常數名稱
* self::常數名稱
