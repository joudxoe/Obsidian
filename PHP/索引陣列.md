定義空陣列
```
$name=array();
```

```
$name=[];
```

定義索引陣列
```
$name[]='Jane';
$name[]='Sam';
```

```
$name[0]='Jane';
$name[1]='Sam';
```

```
$name=array('Jane','Sam');
```

```
$name=array(0=>'Jane',1=>'Sam');
```

PHP5.4加入使用方括號新增陣列元素的語法
```
$name=['Jane','Sam'];
```

要點整理
-  索引陣列 (Indexed Array) - 以==數字==為索引
