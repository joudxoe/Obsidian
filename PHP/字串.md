PHP字串輸出方法
* echo
* print
* printf()
* sprintf()
* print_r()
* var_dump()
* var_export()

printf() - 格式化輸出字串
```
printf('%s','Jane');	// 字串
printf('%d',100);		// 包含正負號的十進制數(負數、0、正數)
printf('%u',100);		// 不包含正負號的十進制數(大於等於0)
printf('%f',100);		// 浮點數
printf('%.3f',100);		// 小數點後3位
printf('%e',100);		// 小寫的科學計數法
printf('%E',100);		// 大寫的科學計數法
```

sprintf() - 將輸出字串指派給變數
```
$str=sprintf('%s(%d)','Jane',22);
```

補充資料

1. 如果不需要解義(evalute)變數和跳脫字符，建議使用單引號，有文件指出效能會比較好
2. PHP會解譯雙引號字串中的變數名稱
