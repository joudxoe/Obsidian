PHP字串表示法
* 單引號表示法 (Single Quoted)
* 雙引號表示法 (Double Quoted)
* Heredoc
* Nowdoc

Heredoc
```
$str=<<<EOD

EOD;
```

Nowdoc
```
$str=<<<'EOD'

EOD;
```

在雙引號中使用關聯陣列，須再加上大括號，才能正常解譯其值
```
$author=['name'=>'Jane'];
echo "Author: {$author['name']}";
```

要點整理
- 如果不需要解義(evalute)變數和跳脫字符，建議使用單引號，有文件指出效能會比較好
- PHP會解譯雙引號字串中的變數名稱
