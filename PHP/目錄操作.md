要點整理
- 目錄是一種特殊的文件
- 瀏覽目錄下的文件，首先要打開目錄，瀏覽完後同樣要關閉目錄

打開目錄
```
$path='C:/Users/TPDNXU/Downloads';
$dir=opendir($path);
```

瀏覽目錄
```
$path='C:/Users/TPDNXU/Downloads';
$dir=scandir($path);
foreach($dir as $file){
	printf('%s<br>',$file);
}
```

```
$path='C:/Users/TPDNXU/Downloads';
$dir=opendir($path);
whlie(($file=readdir($dir))!==false){
	printf('%s<br>',$file);
}
```

關閉目錄
```
closedir($dir);
```
