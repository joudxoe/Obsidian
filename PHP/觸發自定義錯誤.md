自定義錯誤處理函數
```
function customError($errno,$error){
	printf('<b>Error:</b> [%d] %s',$errno,$error);
	die();
}
```

設定錯誤函數的處理
```
# 錯誤類型預設為E_ALL

# set_error_handler('customError',E_USER_WARNING);
```

觸發錯誤
```
$password='admin123';

if($password != 'password'){
	trigger_error('wrong password',E_USER_WARNING);
}
```

錯誤訊息
```
# Error: [512] wrong password
```
