使用foreach遍歷/循環陣列
```
foreach($arr as $val){

}
```

```
foreach($arr as $key=>$val){

}
```

巢狀式foreach遍歷多維陣列
```
foreach($arr as $subarr){	
	foreach($subarr as $val){

	}
}
```

利用while迴圈迭代陣列
```
while($element=each($arr)){
	printf('%d:%s',$element['index'],$element['value']);
}
```

```
while(list($key,$val)=each($arr)){

}
```
