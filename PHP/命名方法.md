命名方法
- ==駝峰式==命名法 - 第一個字母小寫，之後的單詞第一個字母使用大寫
- ==帕斯卡==命名法 / 大駝峰式命名法 - 所有單詞的首字母都使用大寫
- ==匈牙利==命名法 - 在名稱前加上前綴，例如類型的縮寫，讓變數名更好理解
- 下劃線命名法 - 單詞字間使用下劃線做區隔

---

駝峰式命名法
```
# $myAge

# $myName
```

帕斯卡命名法
```
# $MyAge

# $MyName
```

凶牙利命名法
```
# i表示整數

# $iMyAge

# f表示浮點數

# $fMyHeight

# s表示字串

# $sMyName
```

下劃線命名法
```
# $my_age

# $my_name
```
