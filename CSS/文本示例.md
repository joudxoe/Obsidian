行高指定百分比值
```
# font-size:10px;line-height:150%;

# 行高等於10px*150%=15px
# font-size:24px;

# 行高等於10px*150%=15px
# font-size:14px;
```

行高指定數值
```
# font-size:10px;line-height:1.5;

# 行高等於24px*1.5=36px
# font-size:24px;

# 行高等於14px*1.5=21px
# font-size:14px;
```

```
# 行距等於2.5*16px=40px
# font-size:16px;
# line-height:2.5em;
```

文本應用
```
# 模糊文本
# color:transparent;
# text-shadow:0 0 5px rgba(0,0,0,0.5);
```

```
# 文本超出省略/隱藏溢位文字/文本截斷
# overflow:hidden;
# white-space:nowrap;
# text-overflow:ellipsis;
```

```
# 兩端對齊
# text-align:justify;
# text-align-last:justify;
```
