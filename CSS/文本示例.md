行高指定百分比值
```
.content{
	font-size:10px;
	line-height:150%;
}
h1{
	font-size:24px;		// 行高等於10px*150%=15px	
}
p{
	font-size:14px;		// 行高等於10px*150%=15px
}
```

行高指定數值
```
.content{
	font-size:10px;
	line-height:1.5;
}
h1{
	font-size:24px;		// 行高等於24px*1.5=36px	
}
p{
	font-size:14px;		// 行高等於14px*1.5=21px
}
```

```
p{
	font-size:16px;
	line-height:2.5em;		// 行距等於2.5*16px=40px
}
```
