```
# 可解決圖片底部的空白問題，可清除空白節點/空白區域
# .parent{font-size:0;}
```

```
# 在所有瀏覽器顯示相同的文本大小
# body{font-size:100%;}
```

```
# 可防止圖片撐破容器
# img{max-width:100%;}
```

```
# 禁用文本選中，頁面中的文本不能選取
# body{user-select:none;}
```

```
# 禁用事件/任何點擊事件將無效
# .disabled{pointer-events:none;}
```

```
# 響應式圖片
# img{max-width:100%;height:auto;}
```

```
# 改變輸入框焦點光標的顏色
# input,textarea{caret-color:red;}
```
