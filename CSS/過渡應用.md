```
transition-timing-function:cubic-bezier(.20,.96,74,.07); // 定義貝茲曲線
```

```
transition-delay:.5s;	// 動畫轉場前等待半秒鐘
```

```
transition:all 1s linear .5s;	// 所有CSS屬性在延遲半秒後使用linear時間函式轉場一秒
```

---

簡單過渡
```
button {
	border: 1px solid #dc5801;
	transition-property: border;
	transition-duration: 1s;
}
```

```
button:hover {
	border-color: #288049;
}
```

```
button{
	transition-property:all;
	transition-property:background-color,color;
	transition-duration;.5s;			// 動畫運行完成的時間半秒
	transition-duration:500ms,1s;		// 背景色轉場半秒，前景色轉場1秒
}
button:hover{background-color:red;color:white;}
```

多個過渡
```
button {
	background-color: #fd7c2a;
	border: 1px solid #dc5801;
	transition-property: background-color, border;
	transition-duration: 2s, 1s;
}
```

```
button:hover {
	background-color: #3cc16e;
	border-color: #288049;
}
```

過渡簡寫
```
button {
	background: #fd7c2a;
	transition: background 2s ease-in 0s;
}
```

```
button:hover{
	background-color: #3cc16e;
}
```
