```
# 徑向漸變語法

# radial-gradient(shape size at position,color1,color2...);
```

```
# 顏色均勻分佈的徑向漸變

# radial-gradient(red,yellow,green);
```

```
# 顏色均勻分佈的徑向漸變

# radial-gradient(#e66465,#9198e5);
```

```
# 顏色依指定比例分佈的徑向漸變

# radial-gradient(red 5%,yellow %10,green 20%);
```

```
# 圓形徑向漸變

# radial-gradient(circle,red,yellow,green);
```

```
# 楕圓形徑向漸變

# radial-gradient(ellipse,red,yellow,green);
```

```
radial-gradient(closest-side,#3f87a6,#ebf8e1,#f69d3c);
```

```
# 設定漸變結束的大小

# radial-gradient(circle farthest-side at left bottom,red,yellow);
```

```
radial-gradient(circle at 100%,#333,#333 50%,#eee 75%,#333 75%);
```

```
# 重覆徑向漸變

# repeating-radial-gradient(red,yellow 10%,green 20%);
```

```
# radial-gradient(ellipse at top,#e66465,transparent),

# radial-gradient(ellipse at bottom,#4d9f0c,transparent);
```
