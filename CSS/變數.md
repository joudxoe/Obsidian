要點整理
- `var()`函數用於定義CSS變數的值
- CSS變數的作用域有==全局==和==局部==
- 全局變數可以在整個文檔中進行訪問
- 局部變數只可以在聲明它的選擇器內部使用
- 變數名稱必須以兩個破折號(`--`)開頭，且區分大小寫
- 局部變數的值會覆蓋全局變數的值

---

聲明CSS全局變數
```
:root{
	--blue:#6495ed;
	--white:#faf0e6;
}
```

定義CSS局部變數
```
body{
	--blue:#6495ed;
}
```

使用CSS變數
```
body{background-color:var(--blue);}
span{color:var(--white);}
```
