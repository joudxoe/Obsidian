HSL顏色表示法
```
# .red{background-color: hsl(360,100%,50%);}

# .green{background-color: hsl(120,100%,50%);}

# .lightgreen{background-color: hsl(120,100%,75%);}

# .darkgreen{background-color: hsl(120,100%,25%);}

# .softgreen{background-color: hsl(120,60%,70%);}

# .blue{background-color: hsl(240,100%,50%);}

# .lightblue{background-color: hsl(240,100%,75%);}

# .darkblue{background-color: hsl(240,100%,25%);}

# .white{background-color: hsl(0,100%,100%);}

# .black{background-color: hsl(0,100%,0%);}
```

HSLA顏色表示法
```
# 帶不透明度的紅色

# .red{background-color: hsla(360,100%,50%,0.3);}

# 帶不透明度的綠色

# .green{background-color: hsla(120,100%,50%,0.4);}

# 帶不透明度的藍色

# .blue{background-color: hsla(240,100%,50%,0.7);}
```
