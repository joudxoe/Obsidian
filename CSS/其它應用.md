標用鼠標 / 點擊事件無效
```
.disabled{pointer-events:none}
```

黑白圖像
```
img{filter:grayscale(100%)}
```

背景圖居中
```
background-position{center}
background-position{center center}
background-position{50% 50%}
```

防止圖片撐破容器
```
img{max-width:100%}
```

顯示A標籤的連結
```
a[href]:after{content:"attr(href)"}
```

動態切換圖片
```
img:hover{content:url(new.jpg)}
```
