屬性選擇器
- [attribute]
- [attribute=value]
- [attribute~=value]
- [attribute|=value]
- [attribute^=value]
- [attribute$=value]
- [attribute*=value]

```
[target]			// 選取有target屬性的元素
```

```
[target=_blank]		// 選取target="_blank"的元素
```

```
[title~="news"]		// 選取title屬性值包含news的元素
```

```
a[href^="https"]	// 選擇https開頭的連結
```

```
a[href$=".pdf"]		// 選擇PDF檔案的連結
```

```
[lang|="en"]		// 選取lang屬性值以en開頭的元素
```

```
a[href*="google"]	// 選取href屬性值包含google的a元素
```
