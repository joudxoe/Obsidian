```
.intro			// 選取類名為intro的元素
```

```
.intro.blue		// 選取class屬性中同時有intro和blue的元素
```

```
.container .intro		// 選取類名為container且後代類名為intro的元素
```

```
#main		// 選取ID名為main的元素
```

```
p.intro		// 選取類名為intro的p元素
```

```
b,strong		// 選取b元素和strong元素
```

```
div>p		// 選取父元素為div的p元素
```

```
div p		// 選取div元素內的所有p元素
```

```
div+p		// 選取和div元素相鄰的p元素
```

```
p~ul		// 選取和p元素同層的所有ul元素
```
