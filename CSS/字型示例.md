使用em設定字型大小
```
html{font-size:10px;}
.parent{font-size:1.2em;}		// 10px*1.2=12px
.child{font-size:1.2em;}		// 12px*1.2=14.4px
```

```
h1{font-size:2em;}				// 2*16px=32px
h1{font-size:0.875em;}			// 0.875*16px=14px
```

```
body{font-size:62.5%;}				// 預設16px的62.5%，相當於10px=0.625em
h1{font-size:1em;}					// 1*10px=10px
h1{font-size:1.6em;}				// 1.6*10px=16px
p::first-letter{font-size:3em;}		// 3*10px=30px
```

使用rem設定字型大小
```
html{font-size:62.5%;}			// 16px*62.5%=10px
.parent{font-size:1.2rem;}		// 10px*1.2=12px
.child{font-size:1.2rem;}		// 10px*1.2=12px
```
