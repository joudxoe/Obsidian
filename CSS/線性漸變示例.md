```
linear-gradient(#e66465,#9198e5);				// 從上到下的線性漸變
```

```
linear-gradient(to right,red,yellow);			// 從左到右的線性漸變
```

```
-webkit-linear-gradient(left,red,yellow);		// 從左到右的湶性漸變(For Safari)
```

```
-ms-linear-gradient(left,red,yellow);			// 從左到右的線性漸變(For IE)
```

```
linear-gradient(to bottom right,gray,black);	// 從左上到右下的對角線性漸變
```

```
-webkit-linear-gradient(top left,gray,black);	// 左上到右下的對角漸變(For Safari)
```

```
-ms-linear-gradient(top left,gray,black);		// 左上到右下的對角漸變(For IE)
```

```
linear-gradient(90deg,white,gray,black);		// 從左到右帶有多個顏色的線性漸變
```

```
linear-gradient(red,orange,yellow,green,blue,indigo,violet); 	// 彩虹線性漸變
```

```
linear-gradient(red,yellow 30%,lime 60%);		// 使用百分比設定色標停止的位置
```

```
linear-gradient(to right,rgba(255,0,0,0),rgba(255,0,0,1));	// 帶透明度的線性漸變
```

```
repeating-linear-gradient(red,yellow 10%,green 20%);		// 重覆線性漸變
```
