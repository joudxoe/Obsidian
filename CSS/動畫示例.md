定義關鍵畫格
```
@keyframes fadeIn{
	from{opacity:0;}
	to{opaticy:1;}
}
```

```
@keyframes fadeOut{
	0%{opacity:1;}
	100%{opacity:0;}
}
```

```
@keyframes fadeOut{
	0%{opacity:1;}
	25%{opacity:.75;}
	50%{opacity:.5;}
	75%{opacity:.25;}
	100%{opacity:0;}
}
```

套用動畫到元素
```
animation-name:fadeIn;
animation-duration:1s;
animation-timing-function:ease-in-out;
animation-delay:500ms;
```

```
animation: fadeIn 1s ease-in-out 500ms;		// 動畫簡寫屬性
```
