```
# 設定框架開啟的頁面地址

# <iframe src="URL"></iframe>
```

```
# 設置框架的寬度與高度

# <iframe width="800" height="600"></iframe>
```

```
# 不顯示框架邊框

# <iframe frameborder="0"></iframe>
```

```
# 不顯示捲軸

# <iframe scrolling="no"></iframe>
```

```
# 瀏覽器不支援框架

# <iframe>抱歉，您的瀏覽器不支援框架!</iframe>
```

```
# 設置框架名稱

# <iframe src="URL" name="framea"></iframe>

# 在指定框架內開啟外部連結

# <a href="URL" target="framea">URL</a>
```
