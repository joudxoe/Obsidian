無序列表
```
<ul>
	<li>ItemA</li>
	<li>ItemB</li>
</ul>
```

有序列表
```
<ol>
	<li>ItemA</li>
	<li>ItemB</li>
</ol>
```

嵌套列表
```
<ul>
	<li>ItemA
		<ol>
			<li>ItemA.1</li>
			<li>ItemA.2</li>
		</ol>
	</li>
	<li>ItemB</li>
</ul>
```

自定義列表
```
<dl>
	<dt>Firefox</dt>
	<dd>A open-source web browser developed by the Mozilla Corporation.</dd>
</dl>
```
