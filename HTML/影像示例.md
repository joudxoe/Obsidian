定義視頻
```
<video controls>
	<source src="movie.mp4" type="video/mp4">
	<source src="movie.ogg" type="video/ogg">
	Your browser does not support video element		// 不支持video元素的替代文本
</video>
```

```
<video controls src="movie.mp4">你的瀏覽器不支援video標籤</video>
```

兼容性處理
```
<video controls width="320" height="240">
	<source src="movie.mp4" type="video/mp4">
	<source src="movie.ogg" type="video/ogg">
	<source src="movie.webm" type="video/webm">
	<object data="movie.mp4" width="320" height="240"> // 不支持video元素時用
		<embed src="movie.mp4" width="320" height="240"> // 不支持object元素時用
	</object>
</video>
```
