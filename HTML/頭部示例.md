```
<meta charset="utf-8">	// 正確設定/聲明網頁使用的編碼，防止中文網頁內容出現亂碼
```

```
<meta name="description" content="網頁簡短描述">	// 定義頁面的描述
```

```
<meta name="keywords" content="HTML,CSS,XML">	// 定義頁面的關鍵字
```

```
<meta name="author" content="作者姓名">		// 定義頁面的作者
```

```
<meta name="copyright" content="網頁版權">		// 定義頁面的版權
```

```
<meta name="distribution" content="網頁發佈地區">
```

```
<meta name="robots" content="noindex">		// 指示搜尋引擎不要索引這個頁面
```

```
<meta name="robots" content="nofollow"> 	// 指示搜尋引擎不要追蹤頁面上的連結
```

```
<meta name="robots" content="none">	// 指示搜尋引擎不要索引頁面也不要追蹤網頁上的連結
```

```
<meta name="robots" content="noindex, nofollow">	// 不索引頁面，不追蹤連結
```

```
<meta name="viewport" content="width=device-width, initial-scale=1">
```

```
<meta http-equiv="Content-Type" content="text/html; charset=utf-8"> // HTML4
```

```
<meta http-equiv="Content-Language" content="zh-TW">	// 設定網頁使用的語言
```

```
<meta http-equiv="Refresh" content="time">			// 設定自動更新網頁的時間
```

```
<meta http-equiv="refresh" content="3; url=URL">	// 3秒後自動跳轉到指定網頁
```

```
<meta http-equiv="Pragma" content="no-cache">		// 禁止瀏覽器使用快取開啟網頁
```

```
<base href="URL" target="_blank">		// 定義所有鏈接的默認連結地址
```

```
<link rel="shortcut icon" href="favicon.ico">	// 設置瀏覽器地址欄左側顯示的圖標
```

```
<link rel="bookmark" href="favicon.icon">	// 設置收藏夾中顯示的圖標
```
