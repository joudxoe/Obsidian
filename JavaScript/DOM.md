DOM屬性

*`innerHTML` 獲取元素的內容*
*`parentNode` 元素的父節點*
*`childNodes` 元素的子節點*
*`attributes` 元素的屬性節點*

DOM方法

*`getElementById()` 返回帶有指定ID的元素*
*`getElementsByTagName()` 返回帶有指定標籤名的元素列表*
*`getElementsByName()`*
*`getElementsByClassName()`*
*`appendChild()` 插入新的子節點*
*`removeChild()` 刪除子節點*
*`replaceChild()`*
*`insertBefore()`*
*`createAttribute()`*
*`createElement()`*
*`createTextNode()`*
*`getAttribute()`*
*`setAttribute()`*

***

取得文檔中A標籤的數量
```
document.anchors.length
```

返回文檔中的連結數(具有href屬性)
```
document.links.length
```

返回文檔中圖片的數量
```
document.images.length
```

返回文檔的URL
```
document.URL
```
